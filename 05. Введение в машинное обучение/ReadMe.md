# Рекомендация тарифов

## Цель проекта
Построить модель для задачи классификации, которая выберет подходящий для пользователя тариф.

## Вывод

Наилучшее качество показал **GradientBoostingClassifier** с параметрами:
* learning_rate=0.0785
* max_depth=2
* min_samples_leaf=16
* max_features=0.618
* n_estimators=234
